# Holberton School - Higher Level Programming

This repository contains projects and exercises for higher level programming concepts, primarily focused on Python programming and C low-level programming.

## Projects

### Python Import Modules
- **Directory**: `python-import_modules/`
- **Description**: Learning to import functions and modules in Python
- **Files**: 0-add.py through 5-variable_load.py

### Holberton Printf
- **Directory**: `holbertonschool-printf/`
- **Description**: Custom implementation of the printf function in C
- **Files**: _printf.c, main.c, main.h

### Simple Shell
- **Directory**: `holbertonschool-simple_shell/`
- **Description**: Implementation of a simple UNIX command line interpreter
- **Files**: execute.c, get_line.c, main.c, shell.h

### Sorting Algorithms
- **Directory**: `holbertonschool-sorting_algorithms/`
- **Description**: Implementation of various sorting algorithms in C
- **Files**: bubble_sort.c, insertion_sort_list.c, selection_sort.c, quick_sort.c
